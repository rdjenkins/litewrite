Privacy statement

Litewrite does not have access to any of your data or store any authentication data to your remote storage provider - you need to authenticate directly with them.

Data is either stored in your browsed in local storage or are available everywhere, using the open remoteStorage standard, Dropbox or Google Drive.

Litewrite is an unhosted web app, meaning its code is fully client-side, without any server backend you need to trust! It also supports the open remotestorage protocol so you can sync your data across devices & browsers.

For more information go to Litewrite on GitHub.
